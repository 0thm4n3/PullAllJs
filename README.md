# PullAllJs
An automated tool that pulls and organizes All JS files from webpages using the TomNomNom's (WayBackUrls)[https://archive.org]


# Install
- `git clone https://github.com/0thm4n3/PullAllJs`
- `cd PullAllJs`
- `chmod +x *`
- `./install`

# Usage
- `./pulljs.sh <DONMAIN.COM/IP>`
